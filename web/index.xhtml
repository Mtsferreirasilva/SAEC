SAEC

Sistema de Agendamento de Exames Clínicos

Garantindo o maior conforto e melhor atendimento.

Profissionais especializados e bem preparados.

Equipamentos modernos e variados.

Conforto durante o atendimento.

O SAEC é um software online onde é possível realizar o agendamento de exames, bem como salvar o diagnóstico após realização destes, cadastrar e gerenciar pacientes, médicos e exames, assim como obter relatórios das consultas prestadas.

É um software livre desenvolvido utilizando HTML5, CSS3 e o framework Twitter Bootstrap na camada de apresentação, JSF e JPA na camada de negócios e o SGBD MySQL na camada de acesso a dados.
